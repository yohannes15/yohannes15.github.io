A purely functional chess engine and backend in Scala 3, implementing move generation, validation, FEN/PGN parsing, and AI opponent with alpha-beta pruning.

Currently building the HTTP API layer with Http4s for move validation and best-move suggestions. Future plans include WebSocket multiplayer, game persistence, and a web UI. Inspired by Lichess's scalachess library.
